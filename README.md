### Навыки:
+ Анализ продуктовых метрик с помощью SQL(JOIN, подзапросы, оконные функции) в PostgreSQL, SuperSet Lab, Redash.
+ Визуализация данных в Apache SuperSet, Power BI - создание, актуализация и поддержка работоспособности дашбордов с ключевыми аудиторными метриками и событиями продукта за прошедшее время, анализ аудитории по характеру использования приложения, анализ эффективности рекламных кампаний.
+ Построение ETL-пайплайна в Airflow, автоматизация отчетности и автоматический поиск аномалий с отправкой отчетов через Telegram Bot
+ Использую Python для анализа данных (pandas, numpy, matplotlib, seaborn), оформляю результаты в Jupyter Notebook
+ На основании проведения A/B тестирования могу сделать выводы о продукте.
